# L01 - Linear Regression (Part I)

## Agenda
* Introduction: overview of the course content and structure.
* Brief introduction to `std::vector` (used throughout the course).
* What is machine learning?
* Linear regression: prediction, parameters, error, epochs, and learning rate.
* Implementing the linear regression model in software, and training it.

---

## Objectives
* Know how `std::vector` is used in C++.
* Know why a traditional program isn't well suited for mimicking human intelligence.
* Know how machine learning differs from traditional programming in terms of input, output, and rules.
* Be able to train simple linear regression models by hand.
* Be able to implement training of a linear regression model in software.

## Evaluation
* What's the difference between input, rules, and output in a traditional program compared to a machine learning system?
* What's meant by an epoch when training a machine learning model?
* How are the parameters in a linear regression model used to compute a prediction for a given input?
* Why is an interface used for the linear regression model? What does it enable?
* What happens to the model's weight and bias value when input `x` equals zero during training?
* How does the learning rate affect training? What happens if it's set too high or too low?
