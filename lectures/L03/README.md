# L03 - Neural Networks (Part I)

## Agenda
* Introduction to neural networks: feedforward, backpropagation, gradient descent, and activation functions.
* Training neural networks by hand.
* Starting an exercise: creating a dense layer interface and a stub implementation of it.

---

## Objectives
* Know what a neural network is and be able to explain concepts such as layers, nodes, and weights.
* Be able to train small neural networks by hand.
* Have created an interface for a dense layer and a stub implementation of it.

---

## Instructions

### Preparation
* Read through [appendix A](./appendix/a_theory.md), covering neural network theory and hand-training, plus an overview of the dense layer interface you'll be building.

### During the Lecture
* Participate in the lecture (live coding session).
* Hand-training of neural networks is covered during the lecture.

### After the lecture
* Complete the exercises in [appendix B](./appendix/b_exercises.md).
* Solutions will be available a few days after the lecture.

---

## Evaluation
* Can you, in your own words, explain what happens in a neural network during feedforward and backpropagation?
* What's the purpose of an activation function, and what happens if you don't use one?
* What happens to training if the learning rate is set too high or too low?
* Why is it beneficial to define an interface for the dense layer rather than implementing a class directly?
